# 🔍 Haystack Search App

Cette application permet d'effectuer une **recherche intelligente de mots-clés** ou de **questions** dans vos documents (`.pdf`, `.docx`, `.txt`) en utilisant le moteur Haystack (IA basée sur RoBERTa).
